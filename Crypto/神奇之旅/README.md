# 神奇之旅

## 題目

```
從前從前，有個神秘的商人，他對一切與古代文明有關的事物充滿熱情，特別是那些充滿謎團的物品。有一天，他在法國發現了一張羊皮紙，上面寫：「 ÔQÔÎŒÔ{Îwvpzfœaûfœrwvæ, ewa fzk wvk æœky îûwækz œw yfæærvp œqk kdfü.} 」，他立刻意識到這並非普通的書寫，而是一段用密碼加密的文字。這讓他興奮不已，因為這樣的發現通常意味著背後隱藏著驚人的秘密。密碼的來源並不清楚，但羊皮紙的邊角處有一行法文：“L'héritage de César.”這幾個字讓他心跳加速。他立刻想到，這可能是羅馬帝國偉大統治者César大帝留下的密碼！他開始研究，發現這串密碼似乎與某種密碼有關，這是一種古老的加密方法，通過字母的位移來隱藏訊息。
現在請你幫他找到這串密碼。

可參考網站：
https://en.wikipedia.org/wiki/Caesar_cipher

```
